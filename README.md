TITLE:
Cashly - Smart Expense Tracking with AI

DESCRIPTION:
An AI-powered expense tracking app that automatically scans and analyzes receipt images using Google Gemini 2.5 Flash. Built with Next.js 16, it features a CRED-inspired design with interactive data visualizations, automatic expense categorization across 9 categories, and real-time spending analytics. Users can upload receipts via camera or file selection and track financial trends with detailed breakdowns.

LINKS:
Live Demo: https://cashly.splin.app

SKILLS/TOOLS:
Next.js 16, React 19, TypeScript, Google Gemini AI, PostgreSQL, Drizzle ORM, Tailwind CSS, Recharts, Vercel AI SDK

CONTRIBUTIONS:
- Built full-stack application with Next.js App Router and PostgreSQL database
- Integrated Google Gemini 2.5 Flash for AI-powered receipt analysis
- Designed CRED-inspired UI with interactive charts and mobile-first responsive layout
- Implemented expense categorization, spending analytics, and trend visualization
- Set up SEO optimization with dynamic metadata and Open Graph images
